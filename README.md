# Landing Page Sapataria Chrispim Vieira

Fundacao tecnica da landing page one-page da Sapataria Chrispim Vieira, criada com Next.js, TypeScript, Tailwind CSS e Docker.

## Rodar localmente

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

Para testar o modo servidor de producao:

```bash
npm run build
npm run start
```

## Rodar com Docker

```bash
docker compose up --build
```

A aplicacao ficara disponivel em `http://localhost:3000`.
